# Council Bronze Layer (SQL Server)

This is a small learning project demonstrating a basic Bronze layer structure in SQL Server.

## Goal
To practice structuring raw financial-style data in a Bronze schema, similar to early-stage data warehouse ingestion.

## Tech
- SQL Server
- T-SQL

## Data Note
The dataset used in this project is synthetic (sample data) created for learning purposes.
It is inspired by the structure of UK local authority open payment datasets but does not represent real transactions.
